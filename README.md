# SchwingerAtomMesonModesEigensolver
A collection of scripts which solve for the free meson eigenmodes of the Schwinger atom theory.

generatingmesonevalsevecs.nb - Calculates the eigenmodes with Dirichlet boundary conditions given g,\kappa, and a steady state soliton.  Exports as .wl files

svdschwinger.nb - Calculates eigenmodes with outgoing BCs using a singular value decomposition

SUSYeigenproblem.nb - Calculates modes of the supersymmetric partner theory, and uses this to calculate classical scattering properties
